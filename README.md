## Rebuilding the Database

### Postgres

#### Build
```
dotnet ef migrations add InitialCreate
dotnet ef database update

```
#### Delete
In Query Editor:
```
DROP SCHEMA public CASCADE;
CREATE SCHEMA public;

```
In Package Manager Console
```
dotnet ef migrations remove
```
